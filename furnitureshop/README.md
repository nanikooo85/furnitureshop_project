## ­ЪЏа№ИЈ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ рЃЊрЃљрЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ Python (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ Python 3.11).

### 1. рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃљ

python -m venv venv
# рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃљ (Windows)
.\venv\Scripts\activate
# рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃљ (macOS/Linux)
